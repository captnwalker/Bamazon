# Bamazon

>An Amazon-like CLI storefront built with mySQL, NODE.js and Inquirer NPM. The app can take in orders from customers and deplete stock from the store's inventory.

### Deployment

1. Clone repo
2. Run `npm install`
3. At command prompt run `node bamazon.js`
